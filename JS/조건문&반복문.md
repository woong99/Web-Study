# 조건문&반복문

## If

```JavaScript
const a = 5;

if (a === 0) {
  console.log('a is 0');
} else if (a === 2) {
  console.log('a is 2');
} else {
  console.log('rest...');   // rest... 출력
}
```

---

## Switch

```JavaScript
const a = 4;

switch (a) {  // a가 조건
  case 0:
    console.log('a is 0');
    break;  // 각 조건이 끝나면 break 필수!
  case 2:
    console.log('a is 2');
    break;
  case 4:
    console.log('a is 4');  // a is 4 출력
    break;
  default:  // 위의 조건들을 제외한 경우, break 필요 없음.
    console.log('rest...');
}
```

## 반복문 (For statement)

### for (시작조건; 종료조건; 변화조건) {}

```JavaScript
for (let i = 1; i < 4; i += 1) {
  console.log(i);
}
// 1
// 2
// 3
```
