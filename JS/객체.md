# 객체

## Object.keys

객체의 key 값 들을 추출

```JavaScript
const user = {
  name: 'Woong',
  age: 24,
  email: 'dnfsss@gmail.com',
};

const keys = Object.keys(user);
console.log(keys);  // ['name', 'age', 'email']

const values = keys.map((key) => user[key]);
console.log(values);    // ['Woong', 24, 'dnfsss@gmail.com']
```
